# Heston-Model-Caliberation-IV-Smile

•	Implemented risk-neutral Heston model (characteristic function + numerical integration) and calibrated to option chains. 
•	Ingested live AAPL chains via yfinance; minimized squared price errors, reported RMSE and calibrated parameters. 
•	Produced diagnostics: market vs model prices, volatility smiles, and 3D/contour implied-volatility surfaces with Matplotlib.
